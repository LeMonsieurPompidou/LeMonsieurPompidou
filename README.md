## Featured Projects

### [Neuromuscular Exoskeleton Adaptation](https://github.com/LeMonsieurPompidou/exoskeleton-adaptation-analysis)
*Master's Thesis conducted at the University of Technology Sydney (UTS) in collaboration with the EPFL Biorobotics Laboratory.*

This research investigated neuromuscular adaptation mechanisms during upper-limb robotic assistance through a multimodal framework combining:
* Surface EMG analysis & Vicon motion capture
* Biomechanical data processing & Human-robot interaction studies

---

### [Terraform Ephemeral VPN](https://github.com/LeMonsieurPompidou/ephemeral-vpn-gateway)
*Personal infrastructure engineering project focused on secure on-demand VPN deployment.*

The system automatically provisions and destroys temporary VPN gateways using Infrastructure as Code (IaC) principles, reducing operational costs and minimizing security exposure.
* **Key Tech:** Terraform, WireGuard, Cloud Infrastructure, Python, DevOps Automation
